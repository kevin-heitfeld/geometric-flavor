## Troubleshooting

### LaTeX Compilation Issues

**Problem:** "File 'physics.sty' not found"
```bash
# TeX Live
tlmgr install physics

# MiKTeX
mpm --install=physics
```

**Problem:** "Undefined control sequence \vev"
- The `physics` package provides `\vev{}` command
- Ensure physics package loads before document begins

**Problem:** Bibliography not appearing
```bash
# Must run in this order:
pdflatex main.tex    # First pass
bibtex main          # Generate bibliography
pdflatex main.tex    # Second pass (include bib)
pdflatex main.tex    # Third pass (fix references)
```

### Python/Figure Generation Issues

**Problem:** "ModuleNotFoundError: No module named 'numpy'"
```bash
pip install numpy scipy matplotlib
```

**Problem:** Figures look pixelated
- PNG files are 300 DPI by default (publication quality)
- Use PDF files for vector graphics (infinitely scalable)
- Check: `dpi=300` in matplotlib savefig calls

**Problem:** "Permission denied" when saving figures
- Ensure `figures/` and `figures/supplemental/` directories exist
- Check write permissions in manuscript directory

### Common LaTeX Warnings

**Warning:** "Overfull \hbox" 
- Line extends into margin slightly
- Usually cosmetic, can ignore unless severe (>5pt)

**Warning:** "Citation X undefined"
- Run bibtex and recompile
- Check reference exists in references.bib

**Warning:** "Reference Y multiply defined"
- Duplicate \label{} commands
- Each label must be unique
